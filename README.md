# 使用说明
```bash
conda create -n face1n1 python=3.10.12
conda activate face1n1
cd tools/
bash env_config.sh
cd ..
python pyfat_implement.py
tar -cvf 1n1_ubuntu_cu12_<版本号>.tar ./
```

5. 在浏览器登录视重通测试平台，网址：`11.33.4.180:8083/login`，账号为 `sys`，密码为 `111111`。
